# odin-recipes

##Overview

Odin project is a simple website that I'm going to build as part of the TOP foundation curriculum.
it will demonstrate what i've learned so far by creating a multi-page recipe website with links between pages.

The recipe information and cooking instructions were inspired by and referenced from the official Knorr Philippines Chicken Adobo recipe.
